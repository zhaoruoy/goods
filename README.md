# 饮品管理系统

本项目是一个基于前后端分离架构开发的饮品管理系统，旨在实现用户管理、饮品管理、饮品类别管理等功能模块。系统提供用户的注册、登录和安全退出功能，并实现统一的权限验证。

## 技术栈

- **前端开发**：
  - **框架**：Vue.js
  - **UI库**：ElementUI
  - **HTTP客户端**：Axios

- **后端开发**：
  - **框架**：Spring、SpringBoot
  - **ORM框架**：Mybatis
  - **编程语言**：Java
  - **JDK版本**：1.8

- **数据存储**：
  - **数据库**：MySQL

## 功能模块

- 用户管理
- 饮品管理
- 饮品类别管理

## 安全与权限

- 用户注册
- 用户登录
- 安全退出
- 统一权限验证
